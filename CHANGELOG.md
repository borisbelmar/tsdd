## [0.2.0] - 2026-05-20

### Added

- Skills collection under `skills/`:
  - `tsdd-init` — project initialization, context discovery, docs structure, AGENTS.md setup
  - `proposal-writer` — structured proposal writing for fuzzy problem spaces
  - `spec-writer` — writing and maintaining executable specs
  - `spec-implement` — implementing specs end-to-end
  - `agents-writer` — creating and editing AGENTS.md with project context and harness boundaries
- Examples directory with real-world case studies:
  - Solo developer, personal project (`logtail` CLI)
  - Solo developer, client with backlog (AutoParts Store e-commerce)
  - Small team 2-3 devs (`Reserva` SaaS booking platform)
  - Large team 7-8 devs (`MercadoLocal` multi-vendor marketplace)
- Workflow diagram image in `assets/tsdd-cycle.png`
- Skills section in README with links to each skill
- Examples section in README with links to case studies
- CHANGELOG.md
